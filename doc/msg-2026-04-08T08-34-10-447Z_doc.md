# 交付说明：Workspace 同步 .github 功能

**源任务文件：** `msg-2026-04-08T08-34-10-447Z.md`  
**任务目标：** left_menu 右键 workspace，新增「同步 .github」功能，用源 workspace 的 .github 覆盖目标 workspace 的 .github

## 改动

### `application/server.js`

新增 `POST /api/workspaces/:workspace/sync-github`：
- body: `{ sourceWorkspace: string }`
- 验证源 workspace 存在、有 `.github` 目录
- `rm -rf target/.github` → `cp source/.github → target/.github`
- 返回 `{ synced: true, from, to }`
- 404 error if source workspace or `.github` not found

### `application/public/app.js`

- `ensureSyncGithubModalDOM()` — 懒加载弹窗 DOM（下拉选择来源 workspace）
- `openSyncGithubModal(wsName)` — 打开弹窗，填充来源列表（排除自身）
- `closeSyncGithubModal()` / `submitSyncGithub()` — 关闭/提交逻辑
- 右键菜单新增「同步 .github」项（在「克隆 Workspace」之后）

## 验证

- `node --check` 语法通过（app.js + server.js）

## 风险/阻塞

- 需要重启服务器以激活新 API 端点
