# Task: Fix right-click context menu not visible

## Source task file
`msg-2026-04-09T03-54-12-803Z.md`

## Goal
Right-click context menu on mind map nodes was invisible — user saw nothing on right-click.

## Root cause
`.dm-ctx` (context menu) had `z-index: 999`, but `.modal-backdrop` (the parent modal overlay) had `z-index: 10001`. The context menu was rendering behind the modal.

## Changes made
- **`application/public/styles.css`**: Changed `.dm-ctx` z-index from `999` to `10002` (above modal backdrop's 10001).

## Verification
- CSS inspection confirms z-index stacking: modal-backdrop (10001) < dm-ctx (10002) < toast (20000).

## Remaining risks / blockers
None.
