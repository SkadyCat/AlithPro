## 主界面布局分析

当前仓库没有 Web 前端或桌面 GUI 文件，因此“主界面”实际是 `launch-agent.ps1` 构建出的命令行工作界面，布局可分为以下几部分：

- 顶部标题栏：窗口标题会被设置为 `Agent-{Workspace}`，用于标识当前工作区。
- 启动横幅：控制台启动后先输出 `爱丽丝 Agent | workspace: ...` 和日志路径，作为主界面的头部信息区。
- 主输出区：Copilot CLI 的标准输出和错误输出会实时流式显示在控制台中，这是操作过程中最主要的信息区域。
- 状态持久化区：运行输出会同步清洗并写入 `logs\agent-realtime.log`，形成可追踪的后台界面记录。
- 文件队列区：虽然不直接渲染在界面上，但整个交互流程围绕 `sessions\`、`inprocess\`、`processed\`、`doc\`、`logs\` 这几个目录展开，它们共同组成了 Alith 的任务主流程。
