# 交付说明：PowerShell 可用性测试

## 源任务文件
`msg-2026-04-07T13-46-42-403Z.md`

## 目标
测试 PowerShell 是否可用。

## 结果

**PowerShell (pwsh) 当前不可用。**

所有 `powershell` 工具调用均返回 `File not found: >` 错误，这与上一会话（`session-2026-04-07-21-21-25.md`）的诊断结论一致：目标设备未安装 PowerShell 7（`pwsh.exe`）。

### 已有修复方案

仓库根目录提供一键修复脚本 `install-pwsh.bat`，执行后会通过 `winget` 自动安装 PowerShell 7：
```
install-pwsh.bat
```
安装完成后重新启动 Copilot Agent，PowerShell 工具将恢复正常。

## 影响
- 当前会话所有需要 shell 命令的操作（如文件移动 `Move-Item`）均无法执行；采用虚拟 inprocess/processed 跟踪（通过日志和 doc 记录）代替。
- 代码编辑、文件查看、任务文档处理不受影响。

## 剩余风险
- 若 `winget` 未安装，需手动从 https://github.com/PowerShell/PowerShell/releases 下载安装包。
