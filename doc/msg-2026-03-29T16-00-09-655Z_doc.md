# 交付说明：UMGEoolEx 视觉预览功能

## 源任务文件
`msg-2026-03-29T16-00-09-655Z.md`

## 目标
在 UMG Blueprint Explorer 中添加视觉预览模式（像 UMG Designer 那样显示 Widget 层级结构），而非仅显示节点树。用户举例：Overlay 里面套了一个按钮和文本。

## 改动内容

### `application/workspace/UMGEoolEx/application/public/app.js`

1. **`renderAssetInfo()`**：新增「🖼 视觉预览」按钮（橙黄色主题），与 Export JSON / 生成数据结构并列

2. **`WIDGET_CATEGORY` 常量**：定义 4 类控件及其 CSS 颜色主题
   - `container`（蓝色）：Overlay, CanvasPanel, HorizontalBox, VerticalBox, GridPanel, WrapBox, ScrollBox, SizeBox, Border, WidgetSwitcher, SafeZone, UniformGridPanel
   - `interactive`（绿色）：Button, CheckBox, ComboBoxString, EditableText, SpinBox, Slider…
   - `display`（橙黄）：TextBlock, RichTextBlock, Image, Throbber…
   - `list`（紫色）：TileView, ListView, TreeView

3. **`widgetCategory(className)`**：将控件类名映射到上述 4 类

4. **`widgetLayoutClass(className)`**：根据容器类型返回布局模式
   - `HorizontalBox` → row、`VerticalBox` → col、`GridPanel/UniformGridPanel` → grid、其余 → stack

5. **`buildWidgetPreviewEl(node, depth)`**：递归构建 HTML 预览元素，每个控件显示为带颜色的方框，内含「类名 | 对象名」标签，容器控件自动包含子元素

6. **`showVisualPreview(roots, fileName)`**：展示全屏预览弹层，顶部显示文件名和图例，按 Esc 或点击「✕ 关闭」退出

### `application/workspace/UMGEoolEx/application/public/style.css`

新增 Visual Preview 样式块（`.vp-overlay`、`.vp-header`、`.vp-canvas`、`.vp-widget`、`.vp-label`）：
- 4 种颜色主题：`.vp-container`（蓝）/ `.vp-interactive`（绿）/ `.vp-display`（琥珀）/ `.vp-list`（紫）
- 4 种布局：`.vp-layout-row` / `.vp-layout-col` / `.vp-layout-grid` / `.vp-layout-stack`

## 效果示例

选中一个含 Overlay → Button + TextBlock 层级的蓝图后，点击「🖼 视觉预览」将显示：

```
┌─────────────────────────────────┐  ← 蓝色 (Overlay)
│  Overlay | [name]               │
│  ┌──────────┐  ┌─────────────┐  │
│  │Button    │  │TextBlock    │  │
│  │[name] 绿 │  │[name] 琥珀  │  │
│  └──────────┘  └─────────────┘  │
└─────────────────────────────────┘
```

## 验证结果
- 人工语法检查通过（784 行 IIFE，无报错）
- 按钮正确注册到 `renderAssetInfo()` 的 DOM 事件中
- `showVisualPreview()` 可处理空 roots（无子节点时不渲染 children div）

## 剩余风险
- 无 CanvasPanel Slot 位置信息（`.uasset` 解析器目前不提取 Slot 偏移/对齐），因此 CanvasPanel 的子元素按顺序堆叠，不反映真实坐标
- 超大蓝图（200+ 节点）可能导致预览层内容较长，可通过滚动查看
