# 交付说明 — msg-2026-04-08T07-23-19-670Z

**源任务文件**：msg-2026-04-08T07-23-19-670Z.md（子任务，父任务：msg-2026-04-08T07-14-31-121Z.md）
**任务目标**：hover 到卡片时，同一链路的所有节点闪缩紫色高亮边缘

## 实现说明

功能基础（CSS 动画 + mouseenter/mouseleave 事件）已在前序版本中实现，但存在一个关键 bug 导致 hover 无效。

### Bug 根因

getChainRoot() 通过 DOM 遍历找根节点（查询 data-filename 属性），若父任务卡片因 bucket filter 不可见（未渲染到 DOM），则遍历中止，返回子任务自身 fileName。随后 getChainMembers(child) 只找到 {child} 一个成员，if (members.size < 2) return 直接跳过高亮。

### 修复方案

在 enderBuckets() 里，从 state.currentSummary 的所有 bucket 数据（包括 drafts/sessions/inprocess/processed）预计算：

- state.chainRootMap: Map<fileName, rootFileName> — 每个文件名到其链路根节点的映射
- state.rootsWithSubtasks: Set<fileName> — 作为其他任务父节点的文件名集合

在 uildCard() 中：
- 设置 card.dataset.chainRoot = resolvedRoot（非链路节点为空字符串）
- 只有 item.subtaskParent 非空或自身是根节点时才设置非空 chainRoot

在 highlightChain() 中：
- 读取 hovered 卡片的 data-chain-root
- 直接查询所有 [data-chain-root="<root>"] 并添加高亮类
- 不再依赖 DOM 遍历，父卡片不可见也能正确高亮链路中所有可见节点
