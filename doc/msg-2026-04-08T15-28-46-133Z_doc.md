# 交付说明：移除思维导图功能

## 源任务文件
`msg-2026-04-08T15-28-46-133Z.md`

## 任务目标
从 Alith Web 应用中完全移除思维导图（Mind Map）功能。

## 所做改动

### 1. `application/public/index.html`
- 移除了思维导图面板 HTML（`#mindmap_panel` 及其所有子元素）
- 更新 `center_area` 注释，去掉 "right (mind map)" 描述
- 移除 `center_left` 的 "Left panel" 注释（不再有左右分栏）

### 2. `application/public/app.js`
- 移除了整个 `initMindMap()` IIFE（约 706 行代码，含节点增删改、拖拽、连线绘制、右键菜单、持久化、重排布局等全部逻辑）
- 移除了 `switchWorkspace` 中的 `window._mmSwitchWorkspace?.()` 调用

### 3. `application/public/styles.css`
- 将 `.center-area` 布局从 `grid-template-columns:1fr 1fr` 改为 `display:flex;flex-direction:column`，使 bucket 列表占据全部宽度
- 移除了 `.center-left` 的 `border-right` 分隔线
- 移除了全部思维导图相关 CSS 类（52 行）：`.mindmap-*`、`.mm-node*`、`.mm-edge`、`.mm-context-menu*`、`.mm-rename-*`、`.mm-msg-*`、`.mm-doc-*` 等

## 验证结果
- ✅ `npm test`（`node --check server.js`）通过
- ✅ 全局搜索确认无残留 `mindmap`/`mind-map`/`_mmSwitchWorkspace` 引用
- ✅ 布局改为单列全宽，bucket 列表自动填满中央区域

## 剩余风险
- 用户浏览器 localStorage 中的 `alith:mindmap:*` 键值仍然存在，不会自动清除（无害残留）
