# 交付说明：将草稿替换为思维导图编辑器

**源任务文件**: `msg-2026-04-08T15-54-49-955Z.md`  
**父任务**: `msg-2026-04-08T15-28-46-133Z.md`（移除思维导图功能）

## 任务目标

将"草稿"功能替换为思维导图编辑器。双击草稿卡片时，打开一个 fashionable 的思维导图编辑器，而非之前的 Markdown 文本编辑器。

## 所做改动

### index.html
- 将草稿 bucket 标题从"草稿"改为"思维导图"
- 将"+ 新建草稿"按钮改为"+ 新建导图"
- 将统计标签从"草稿"改为"导图"
- 将草稿模态框内容从 Markdown 编辑器 + 预览面板替换为思维导图画布
- 模态框标题从"草稿编辑器"改为"思维导图"
- 添加操作提示："右键添加节点 · 双击编辑 · Tab 子节点 · Del 删除"

### styles.css
- 移除所有 `.draft-pane`、`.draft-editor-input`、`.draft-preview` 相关 CSS
- 新增思维导图画布样式：`.draft-mm-canvas`（带圆点网格背景）
- 新增节点样式：`.dm-node`（玻璃质感、渐变左侧色条、悬停发光）
- 新增 SVG 边线样式：`.dm-edge`（渐变贝塞尔曲线 + 发光滤镜）
- 新增右键菜单样式：`.dm-ctx`（毛玻璃背景）
- 模态框加宽至 96vw/1400px 以适配画布

### app.js
- 移除 `draftEditorInput`、`draftPreview`、`updateDraftPreview()` 等旧代码
- 新增完整思维导图引擎（~280 行）：
  - **数据模型**：`{nodes: [{id, x, y, title, parentId}], nextId, panX, panY, zoom}`
  - **渲染**：DOM 节点 + SVG 贝塞尔曲线边线，支持增量更新
  - **交互**：右键添加节点、双击编辑、Tab 添加子节点、Del 删除（含后代级联）、拖拽移动、画布平移、滚轮缩放、Ctrl+A 全选
  - **持久化**：数据以 JSON 存储在 draft 文件中，保存时序列化、打开时解析
  - **兼容**：旧 Markdown 草稿打开时自动转为单节点导图
  - **投递**：promote 前自动转换为可读 Markdown 层级列表
  - **6色方案**：根据节点深度循环分配颜色（紫、青、粉、蓝紫、绿、金）
- 所有"草稿"文本替换为"思维导图"/"导图"

## 验证结果

- `app.js` 语法检查（`new Function(code)`）：通过
- 全局 grep 确认零残留"草稿"引用

## 剩余风险

- 思维导图功能仅做前端语法验证，未在浏览器中运行实际交互测试
- 旧版 Markdown 草稿文件打开后自动转为单节点导图，原始 Markdown 排版信息丢失（不可逆）
- promote 转换为 Markdown 时仅输出层级列表，不保留坐标等视觉信息
