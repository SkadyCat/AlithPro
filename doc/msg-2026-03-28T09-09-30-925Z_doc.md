# 交付说明

- 源任务文件：$taskName
- 任务目标：让 pplication\runserver.bat 在启动服务前先结束上一轮仍在占用端口的旧服务进程。

## 所做改动

1. 更新 pplication\runserver.bat。
2. 启动前先解析 PORT（默认 7439），再用 PowerShell 查找该端口上处于 Listen 状态的旧进程 PID。
3. 对查到的旧 PID 逐个执行 Stop-Process -Id <PID>，然后继续执行现有的 
pm start。
4. 保持原有启动入口不变，没有新增额外脚本。

## 验证结果

- 使用临时端口 7441 先启动一个旧 
ode server.js 实例，占用目标端口。
- 之后执行 unserver.bat（同样使用 PORT=7441），控制台输出：[runserver] stopping previous listener PID=3832 on port 7441。
- 新实例随后成功启动，并输出 lith is running on http://localhost:7441，说明旧服务已被结束且新服务成功接管端口。
- 验证进程已清理。

## 剩余风险或阻塞项

- 当前按“监听端口”识别旧实例；如果同端口上运行的是非本服务进程，也会被当作旧实例结束。按本任务语义这是预期行为，但使用时应确保目标端口专用于 Alith 服务。
- 无其它阻塞项。
