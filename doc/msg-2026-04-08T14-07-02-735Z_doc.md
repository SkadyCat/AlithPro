## Implementation

Added canvas panning — drag on empty canvas space to scroll the mind map view.

### Architecture change
Added `<div id="mindmap_scene" class="mindmap-scene">` wrapper inside `#mindmap_canvas`. All nodes and the SVG are placed inside `scene`. Panning applies `transform: translate(panX, panY)` on the scene div (GPU-accelerated via `will-change:transform`).

### Logic
- `panX`, `panY`, `panState` added to mind map state
- `canvas.mousedown` on empty space (canvas/scene/svg target): records pan start
- `document.mousemove`: if `panState` active → updates `panX/panY` + scene transform
- `document.mouseup`: if mouse didn't move (< 3px) → deselect nodes; otherwise pan committed
- Context menu `cx/cy` corrected: `e.clientX - rect.left - panX`
- Workspace switch resets pan to origin
