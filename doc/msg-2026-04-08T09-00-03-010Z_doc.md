# 交付说明：树节点固定小尺寸

**源任务文件：** `msg-2026-04-08T09-00-03-010Z.md`
**任务目标：** 思维导图节点卡片应固定且足够小，不要充满宽度

## 根因

`.chain-tree-wrap` 和 `.chain-tree-children` 默认 `align-items: stretch`，导致内部卡片撑满容器宽度（整列宽），显示很奇怪。

## 改动

**`application/public/styles.css`**

- `.chain-tree-wrap` 新增 `align-items: flex-start`（卡片不再拉伸）
- `.chain-tree-children` 新增 `align-items: flex-start`（子节点同样不拉伸）
- `.card-tree-node` 新增 `width: 220px`（固定节点宽度，符合思维导图风格）

## 效果

`
[📁 链路标题]
[卡片节点 220px] ← 固定宽
  └─ [子卡片 220px]
        └─ [孙卡片 220px]
`

## 验证

CSS 文件语法正常（无 JS 改动）
