# 交付文档 — 代理 Checkbox + run_without_proxy.bat

## 源任务文件
`msg-2026-04-10T06-35-26-171Z.md`

## 任务目标
在「启动 Agent」按钮旁新增一个「代理」复选框；取消勾选时使用 `run_without_proxy.bat` 启动 Agent（不设置代理环境变量）。同时确保所有 workspace 自动生成 `run_without_proxy.bat`。

## 改动内容

### `application/public/index.html` + `application/public/dev.html`
- 在 `#run_agent_button` 前添加 `<label class="proxy-toggle"><input type="checkbox" id="use_proxy_checkbox" checked /> 代理</label>`

### `application/public/app.js`
- 新增 `useProxyCheckbox` DOM 引用
- `runAgentButton` 点击事件：读取 `useProxyCheckbox.checked`，POST body 增加 `useProxy` 字段
- Toast 消息显示代理状态标签

### `application/public/styles.css`
- `.proxy-toggle`：checkbox 标签样式，与按钮行内对齐

### `application/server.js`
- 新增 `buildRunWithoutProxyBatContent(model)`：无代理版 bat 内容
- `syncWorkspaceLaunchers()`：同步生成 `run_without_proxy.bat`
- `POST /run-agent`：接收 `useProxy` 参数，`false` 时传 `--no-proxy` 给 `launch-agent.py`

### `application/launch-agent.py`
- 新增 `build_run_without_proxy_bat_content(model)`
- `normalize_workspace_launchers()`：同步生成 `run_without_proxy.bat`
- `resolve_launcher()`：新增 `use_proxy` 参数，`False` 时优先选择 `run_without_proxy.bat`
- `main()`：解析 `--no-proxy` 命令行标志

## 验证结果
- `node --check` 通过 server.js、app.js
- `python -m py_compile` 通过 launch-agent.py
