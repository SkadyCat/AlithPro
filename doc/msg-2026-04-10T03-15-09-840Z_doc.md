# 交付文档 — 行为树节点类型

## 源任务文件
`msg-2026-04-10T03-15-09-840Z.md`

## 任务目标
为思维导图节点引入行为树四类型系统：Seq（序列）、Sel（选择）、Action（动作）、Con（条件），右键菜单提供「放置节点」子菜单以选择类型。

## 改动内容

### `application/public/mindmap.js`
- **数据模型**：`_dm.nodeTypes` 定义四种类型元数据（label/icon/color）
- **`dmNodeColor()`**：优先按节点 type 返回专属配色，无 type 时回退深度色
- **`dmAddNode()`**：新增 `type` 参数，创建节点时写入 `node.type`
- **节点渲染**：标题前显示类型徽章 `<span class="dm-node-type-badge">`
- **画布右键菜单**：「放置节点 ▸」展开四类型子菜单
- **节点右键菜单**：「添加子节点 ▸」展开四类型子菜单
- **`dmShowCtx()`**：支持 `children` 属性，渲染二级子菜单
- **"+" 悬浮按钮**：点击弹出四类型选择菜单（替代直接添加）
- **Tab 键**：默认创建 Action 类型子节点
- **`dmToMarkdown()`**：输出带 `[Type]` 标签的 Markdown

### `application/public/styles.css`
- `.dm-node-type-badge` / `.dm-type-{seq,sel,action,con}`：类型徽章样式
- `.dm-ctx-has-sub` / `.dm-ctx-sub`：右键菜单二级子菜单样式
