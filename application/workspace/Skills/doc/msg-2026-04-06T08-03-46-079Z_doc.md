# 交付说明 — Lua 数据树可视化

## 源任务
`msg-2026-04-06T08-03-46-079Z.md`

## 目标
在 3220 端口的 Skill Visualizer 网页中新增第三个视图模式「数据树」，展示 MagicWorld Lua 类层级和 15 个技能实例。

## 改动

### server.js
- 新增 `parseLuaFile()` — 正则解析 Lua class 定义、属性赋值、skillName / assetPath
- 新增 `scanLuaTree()` — 递归扫描 `Script/com/ability/node`，构建文件夹 + 文件树
- 新增 `LUA_NODE_ROOT` 常量、`cachedLuaTree` 缓存、`refreshLuaTree()` 刷新函数
- 新增 `GET /api/lua-tree` 端点，返回完整树结构
- `/api/refresh` 端点现在同时刷新 Lua 树缓存

### app.js (前端)
- `viewMode` 扩展为三态：instances → classes → luaTree
- 新增 `renderLuaTreeMenu()` — 以文件夹嵌套形式渲染左侧目录
- 新增 `renderLuaFolder()` — 递归渲染文件夹节点与文件节点
- 新增 `selectLuaNode()` / `renderLuaDetail()` — 点击节点展示详情面板
- 详情面板分「属性」和「元数据」两个区域
- 搜索过滤同时匹配文件名、className、skillName
- 切换按钮循环：实例 → 类定义 → 数据树

## 验证
- `node --check server.js` / `node --check app.js` — 语法通过
- 服务重启成功，`/api/lua-tree` 返回 35 文件、15 技能实例
- VoidSprint 示例节点确认 className、parentClass、skillName、props 全部正确
