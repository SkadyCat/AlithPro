# 交付说明 — 技能链路优化：DA → Lua 数据

## 源任务
`msg-2026-04-06T08-25-42-317Z.md`

## 目标
优化技能释放链路，将技能配置数据从 DA (DataAsset) 切换为 Lua table 数据源。

## 改动

### 新增文件

**`Content/Script/com/ability/skill_node_data.lua`** — Lua 技能数据模块
- 包含 15 个技能的完整属性表（3 主技能 + 11 被动技能 + 1 无效石板）
- 以 AssetID (int64) 为键，值为属性表：skill_name、class_name、ability_type、release_types、speed 等
- 公共 API：
  - `get_ability(asset_id)` — 返回属性的浅拷贝（防止共享表被污染）
  - `get_all()` / `get_primaries()` / `get_passives()` — 批量查询
  - `get_random_primary()` — 返回随机主技能的 AssetID
  - `has_release_type(asset_id, rtype)` — 释放类型检查
- 启动时自动构建 primary_ids / passive_ids 索引

### 修改文件

**`Content/Script/com/components/skill_comp.lua`**

| 位置 | 旧代码 | 新代码 |
|------|--------|--------|
| L7 | — | `local skill_node_data = require('com.ability.skill_node_data')` |
| `set_skill_info()` | `G.SkillManager:GetUMagicAbility(tag_name)` + `ability.AbilityType ~= G.EAbilityType.invalid` | `skill_node_data.get_ability(tag_name)` + `ability.ability_type ~= "invalid"` |
| `init_skill()` | `G.SkillManager:GetRandomPrimarySkill()` | `skill_node_data.get_random_primary()` |
| `extract_ability_data()` | 读 UE 对象属性 (`ReleaseType:Num()`, `GetClass():GetName()`) | 读 Lua 表字段（纯 Lua，无 UE 依赖） |
| `unlock_skill()` | `G.SkillManager:GetUMagicAbility(skill_id)` | `skill_node_data.get_ability(skill_id)` |

### 未修改（按设计保留）

- `skill_comp:do_skill()` 中的 `G.SkillManager:Create()` — 运行时执行需要 UE Actor 系统
- `entity.lua` 中的 `G.SkillManager:DoSkill()` — 子弹生成/被动处理需要 AngelScript UE 对象
- `SkillManager.as` — DA→AbilityMap 保留，供 AS 运行时使用

## 架构说明

```
配置层 (Lua)                        执行层 (AngelScript)
┌──────────────────┐               ┌──────────────────┐
│ skill_node_data  │               │  SkillManager.as │
│  15 技能配置表    │               │  AbilityMap (DA)  │
│  get_ability()   │               │  DoSkill()       │
│  get_primaries() │               │  SpawnBullet()   │
└────────┬─────────┘               └────────┬─────────┘
         │                                  │
    skill_comp.lua                   entity.lua:do_skill()
    - 解锁/配置/UI                   - G.SkillManager:DoSkill()
    - 纯 Lua 数据驱动                - AS 运行时执行
```
