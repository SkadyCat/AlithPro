# 交付说明 — 技能节点特征 Icon 生成

## 源任务
`msg-2026-04-06T16-16-10-274Z.md` — 根据技能特性生成 icon

## 目标
为 projectile-design.html 中的每个技能节点生成基于其特性的可视化图标。

## 改动

### projectile-design.html
1. **Canvas Icon 生成器** (`generateIcon(skill, size)`):
   - 根据元素类型、分类、形态类型决定渐变色
   - 使用 `hashSeed()` 从类名生成唯一种子，确保同一技能每次生成相同 icon
   - 带缓存 (`ICON_CACHE`) 避免重复绘制

2. **16 种独特形状** (`drawShape()`):
   - 辅助石分类: trajectory=箭头, split=爆裂星, element=菱形, speed=闪电, range=准星圈, duration=沙漏, damage=剑, effect=螺旋点, defense=盾牌, control=十字瞄准
   - 主技能形态: projectile=弹头, ground_target=地面准星, persistent_zone=螺旋, melee=交叉剑, summon=幽灵, aura=同心圆, self_buff=上箭头, channeling=射线, dash=冲刺线, trap=锯齿

3. **颜色系统**:
   - 有元素的技能使用元素色 (火=橙, 冰=蓝, 雷=紫...)
   - 辅助石使用分类色 (trajectory=蓝, split=橙, damage=红...)
   - `shiftHue()` 基于种子偏移色相，确保同分类内色彩有微妙差异

4. **集成位置**:
   - 左侧树: 每个技能节点前显示 20px icon
   - 中央详情: 顶部显示 72px 大 icon (带圆角边框)

## 验证
- ✅ 页面正常加载 (200, ~33KB)
- ✅ generateIcon, drawShape, drawStar, shiftHue, ICON_CACHE, detail-icon 全部存在

## 风险
