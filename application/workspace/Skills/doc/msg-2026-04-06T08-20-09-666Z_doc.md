# 交付说明 — 技能释放链路分析

## 源任务
`msg-2026-04-06T08-20-09-666Z.md`

## 目标
分析游戏内技能系统，从 entity（实体）的 A 键输入到技能释放的完整链路。

## 分析结果

> 注：用户提到的 `entity.lua` 实际为 `entity.as`（AngelScript），`AEntity : AMagicActor` 是空基类。输入处理在 `controller.as`，技能链路核心在 `SkillManager.as`。

### 完整链路总览

```
玩家按键 A → Controller.OnMouseButtonDown → OnBpButtonDown(蓝图)
  → SkillManager.DoSkill(角色, 技能槽index, 动画消息)
    → 读取 character.skill_comp_skill_data[index]
    → 提取主技能: skillLinks[0].type (AssetID)
    → AbilityMap[AssetID] → UMagicPrimary
    → SpawnBullet(能力配置, 位置, 方向) → 生成 ABullet Actor
    → 遍历 PrimaryAbility.AbilityClasses → Process(修改子弹属性)
    → 遍历 skillLinks[1..N] → 被动技能逐一 Process(子弹)
      → 子弹飞行(ProjectileMovement)
        → 碰撞 → OnCompOverlap → 伤害/效果
```

### 各环节详细分析

#### 1. 输入绑定 — `controller.as`
- **文件**：`Script/com/bp/controller.as`
- `SetUpInput()` 将 `KeyA` 绑定到 `OnMouseButtonDown`
- `OnMouseButtonDown(Key "A")` → `OnBpButtonDown("A")`（蓝图事件）
- 蓝图中路由到角色/SkillManager

#### 2. 角色数据 — `BaseMasgicCharacter.as`
- **文件**：`Script/com/bp/BaseMasgicCharacter.as`
- 属性 `skill_comp_skill_data: FSkillDataList`（已装备技能数据）
- 结构：`FSkillDataList → TArray<FSkillDataInfo> → TArray<FPoint2D> skillLinks`
- 每个 `FPoint2D.type` 是 `int64` 类型的 **AssetID**

#### 3. 技能管理器 — `SkillManager.as`（核心）
- **文件**：`Script/com/subsystem/SkillManager.as`
- `OnInit()`：扫描所有 `UMagicAbility` DA，按 AssetID 索引到 `AbilityMap`
- 按 `ReleaseType` 分类：`PrimaryAbilityMap`、`PassiveAbilityMap`
- `DoSkill(Character, index, AnimMsg)`：
  1. 从 `skill_comp_skill_data[index]` 取出技能链
  2. `skillLinks[0].type` → 主技能 AssetID → `UMagicPrimary`
  3. `SpawnBullet(UMagicPrimary, pos, dir)` → 生成 `ABullet`
  4. 设置子弹属性：master、SkillData、OriginalSkillInfo
  5. 遍历 `UMagicPrimary.AbilityClasses` → 各 Process()
  6. 遍历 `skillLinks[1..N]` → 被动技能 Process()

#### 4. 能力定义 — `MagicAbility.as`
- **文件**：`Script/com/ability/MagicAbility.as`
- `UMagicAbility`：基类，含 AssetID、ReleaseType、ElementTypes、Process()
- `UMagicPrimary`：主技能，含 `AbilityClasses`（被动链）、AnimTag
- `UMagicPassive`：被动技能，各子类覆写 Process() 修改子弹

#### 5. 子弹 — `Bullet.as`
- **文件**：`Script/com/bp/Bullet.as`
- `ABullet : AEntity`
- 组件：`NiagaraComp`（特效）、`ProjectileMovement`（物理运动）
- 属性：`SkillData`(AssetID)、`CurrentSkillData`(UMagicBall)、`master`(拥有者)
- `OnCompOverlap()`：碰撞处理 → Behavior.OnOverlap → 伤害结算

#### 6. 被动能力示例 — `PassiveAbility.as`
- **文件**：`Script/com/ability/PassiveAbility.as`
- `UMagicStraitBall.Process()`：设置子弹速度方向
- `UPassiveExplosion.Process()`：添加爆炸事件
- 各子类通过 `Process()` 修改子弹行为

### 关键数据流

| 变量 | 类型 | 用途 |
|------|------|------|
| `AssetID` | int64 | 唯一能力标识，从 DA 配置 → AbilityMap → Bullet |
| `ReleaseType` | EReleaseEnum[] | 技能分类：primary / passive / projectile / wave |
| `skillLinks` | FPoint2D[] | 技能链：[0]=主技能, [1..N]=被动修饰 |
| `master` | int64 | 拥有者实体ID，用于碰撞过滤 |
| `Trail` | UNiagaraSystem | 视觉特效资产 |

### 关键文件索引

| 文件 | 职责 |
|------|------|
| `Script/com/bp/controller.as` | 输入绑定：KeyA → OnMouseButtonDown |
| `Script/com/bp/BaseMasgicCharacter.as` | 角色数据：skill_comp_skill_data |
| `Script/com/subsystem/SkillManager.as` | 技能释放核心：DoSkill + SpawnBullet |
| `Script/com/ability/MagicAbility.as` | 能力定义：UMagicPrimary / UMagicPassive |
| `Script/com/bp/Bullet.as` | 子弹Actor：运动 + 碰撞 |
| `Script/com/ability/PassiveAbility.as` | 被动能力修饰器 |
| `Script/com/bp/entity.as` | 空基类 AEntity : AMagicActor |
| `Script/com/subsystem/EntityManager.as` | 实体管理：ID 分配 + Actor 查找 |
