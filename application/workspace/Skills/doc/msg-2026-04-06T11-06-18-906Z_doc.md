# 在 projectile-design.html 显示新生成的技能图标

**源任务**: `msg-2026-04-06T11-06-18-906Z.md`
**目标**: 将 AIGC 生成的 38 张技能图标集成到 projectile-design.html 页面展示

## 改动

### 文件复制
- 42 张 `skill_*_00001_.png` 从 `AIGC/application/ComfyUI/output/` 复制到 `Skills/application/public/icons/`
- Express 静态服务自动通过 `/icons/` 路径提供

### projectile-design.html 更新

1. **技能行图标**: 每个 skill-row 左侧新增 48×48 缩略图（`iconUrl(en)` helper）
2. **主技能卡片**: 总览页新增 3 张主技能卡片（64×64 图标 + 名称 + 元素标签）
3. **图标画廊**: 总览页底部新增全部 38 张图标的网格预览（64×64 + 中文名）
4. **CSS**: 新增 `.skill-icon-img`、`.primary-grid`、`.primary-card`、`.icon-gallery`、`.icon-tile` 样式
5. **onerror 容错**: 图标加载失败时自动隐藏，不影响布局

### 数据新增
- `primarySkills` 数组：3 个主技能（AbyssalFireRite/DeepSeaFrostSoul/HeavenlyThunder）
- `iconUrl(en)` 函数：`icons/skill_${en}_00001_.png`
