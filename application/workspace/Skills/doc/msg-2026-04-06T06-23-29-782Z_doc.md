# 交付说明：技能节点可视化 Node.js 应用

## 源任务文件
`msg-2026-04-06T06-23-29-782Z.md`

## 任务目标
创建 Node.js 应用，将 MagicWorld 技能节点可视化。左侧菜单列出技能节点，中间区域展示选中节点的配置内容。

## 所做改动

### 新建 `application/` 目录结构
```
application/
  package.json        — 项目配置，依赖 express
  server.js           — Express 服务器 + AngelScript 解析器
  public/
    index.html        — 主页面骨架
    style.css         — 深色主题样式
    app.js            — 前端交互逻辑
```

### server.js 功能
- **AngelScript 解析器**：解析 `.as` 文件中的 class、struct、enum 定义，提取属性（名称、类型、默认值、Category）和方法签名
- **扫描范围**：`ability/`、`bulletbehavior/`、`subsystem/`、`bp/`、`da/`、`da/ui/` 共 153 个节点
- **分类系统**：自动分为 Primary Skills、Passive Skills、Bullet Behaviors、Data Structures、Enums、Subsystems、Entities、Other 8 个分组
- **API 端点**：
  - `GET /api/skills` — 返回全部节点数据和分类树
  - `GET /api/refresh` — 重新解析源文件（支持热更新）
- **端口**：3210

### 前端功能
- **左侧菜单**：按分类折叠展示所有节点，带搜索过滤、分类计数、颜色图标
- **中间区域**：选中节点后显示：
  - 类型标签（Class / Struct / Enum）
  - 源文件路径
  - 继承链（可点击跳转父类/子类）
  - 属性表（名称、类型、默认值、Category）
  - 继承属性（标注来源类名）
  - 方法列表（返回类型、参数、override 标记）
  - 子类列表（可点击跳转）
  - 枚举值网格
