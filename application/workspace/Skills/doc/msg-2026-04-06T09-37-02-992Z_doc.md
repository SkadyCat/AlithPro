## 目标
理解技能释放的同步系统，在关键位置添加运行时日志。

## 同步流程理解

```
[鼠标点击] → entity:receive_input('LeftMouseButton')  [CLIENT]
    ↓
entity:do_skill()  [CLIENT]
    ├ cursor_po = controller():cursor_pos()
    ├ pos, actor = controller():CursorActor()
    └ anim_comp:play_anim('magic_release', {pos=cursor_po}, callback)
        ├ CLIENT: actor:play_anim(anim_name)     → 播放动画
        ├ CLIENT: self.server.play_anim(...)      → RPC 发到服务端
        │   └ SERVER: self.area.do_play_anim(...) → 广播到所有客户端
        │       └ ALL CLIENTS: do_play_anim(...)  → anim_msg = params; play_anim()
        └ CLIENT: 存 tmp_func = callback

[动画事件触发] → AnimEventDelegate → on_anim_callback(name, params, anim_msg)
    └ callback(name, params, custom_params)  [CLIENT]
        ├ params = AnimMsg 结构体（UE 动画系统传入）
        ├ custom_params = anim_msg = {pos: cursor_po}
        ├ 设置 params.Dir / CursorPos / master
        └ skill_comp:do_skill(params)  [CLIENT]
            └ SpawnActor 子弹  [CLIENT 侧]
```

**关键发现**:
- `play_anim` callback 只在发起者客户端执行（`tmp_func` 只存在发起者）
- `do_skill()` 在 CLIENT 执行，`SpawnActor` 在 CLIENT 侧
- `params.ReleasePos` 来自 AnimMsg UE 结构体，entity.lua 未设置此字段
- `custom_params.pos` = 保存的鼠标位置，是可靠的光标位置来源
