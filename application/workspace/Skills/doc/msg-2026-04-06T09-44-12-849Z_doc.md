## 日志分析

### 错误 1: SpawnActor 失败
```
LogSpawn: Warning: SpawnActor failed because the given transform (0,0,0|-90,NaN,NaN|0,0,0) is invalid
```
**原因**: `world:SpawnActor(class, FVector, FRotator)` 在 UnLua 中不能正确传递位置和旋转参数。位置显示 `(0,0,0)`（丢失），旋转出现 NaN。
**修复**: 改用 `UGameplayStatics.BeginDeferredActorSpawnFromClass` + `FinishSpawningActor`，传入 `FTransform(FRotator, FVector, FVector)` 确保位置/旋转/缩放正确。

### 错误 2: play_anim 参数类型错误
```
Error: Invalid parameter type calling ufunction: play_anim, parameter 1: number needed but got no value
```
**原因**: `rush_down` 动画调用时 `params=nil`，传递到 `self.server.play_anim(anim_name, nil)` 时 RPC 序列化失败。
**修复**: `params or {}` — nil 回退为空表。
