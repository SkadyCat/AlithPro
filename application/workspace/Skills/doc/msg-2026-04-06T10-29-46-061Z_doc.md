# 重构 panel_skill_connect.lua 使用 Lua 技能数据

**源任务**: `msg-2026-04-06T10-29-46-061Z.md`
**目标**: 将技能连接面板从 DA 数据（G.SkillManager）切换到 Lua 技能数据（skill_registry）

## 改动文件

### 1. panel_skill_connect.lua — 主面板
- `first_ability()` → `skill_registry.get(info.type)` 替代 `G.SkillManager:GetUMagicAbility()`
- `show_magic_book()` → 新建 Lua 过滤器替代 `G.SkillManager:Filter()`
  - 过滤逻辑：已装备技能不可重复选 + primary/passive 槽位类型匹配
- `on_initialize_skill_slot()` → `skill_registry.get()` + `UE.UObject.Load(icon)` 替代 DA Icon
- `select_skill()` → `.assetId` 替代 `.AssetID`
- 图标加载：`load_icon()` 辅助函数从路径字符串加载 UTexture2D

### 2. panel_skill_screen.lua — 技能书
- `G.SkillManager.AbilityMap` → `skill_registry.get_all()`
- `.SkillName` → `.skillName`
- `.Icon` → `load_icon(skill.icon)`
- `filter:Match(skill, true)` → `filter:match(skill)`

### 3. panel_details.lua — 技能详情
- 新增 Lua table 分支（`type(data) == 'table'`）
- `calc_desc()` 模板替换：`{Key}` → data 对应属性值
- `RELEASE_TYPE_CN` 映射：`primary→主动技能, passive→被动技能, projectile→投射物, topdrop→坠落`
- 保留 UE userdata 兼容分支

### 4. skill_item_widget.lua — 技能列表项
- `G.SkillManager:GetUMagicAbility()` → `skill_registry.get()` + `UE.UObject.Load(icon)`

## Lua 过滤器设计

```lua
build_filter(is_primary_slot)
→ { used_ids = {已装备技能ID集合}, is_primary_slot = bool }
→ match(ability): 未使用 + releaseType 匹配槽位类型
```
