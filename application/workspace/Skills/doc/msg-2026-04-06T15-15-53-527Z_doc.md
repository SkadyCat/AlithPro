# 技能形态类型分类 — MagicWorld 适配方案

## 源任务文件

## 综合分类

综合 Dota 2、Path of Exile、Diablo、Guild Wars 2 等 ARPG/MOBA 的设计模式，梳理出以下主技能形态类型（`releaseType: primary`），以及它们在 MagicWorld 现有体系中的对应关系。

---

### 1. 投射物 (Projectile)
**描述**: 从施法者位置发射一个移动实体，沿方向飞行，碰撞后触发效果。
**游戏实例**: 火球术、冰箭、暗影弹
**MagicWorld 现状**: ✅ 已有 — ABullet + ProjectileMovement，被动链 StraightShot/CurveHoming 控制弹道

### 2. 落点法术 / 地面目标 (Ground Target / Point Target)
**描述**: 选择目标位置，在该位置直接产生效果（无飞行过程）。
**游戏实例**: 落雷 (Lightning Strike)、暴风雪 (Blizzard)、陨石 (Meteor)
**MagicWorld 适配**: 🆕 需新增
- 不需要 ProjectileMovement，直接在 `targetPos` 生成效果
- 可复用 ABullet 容器但速度为 0，或新增专用 Actor
- 参数: targetPos, radius, delay（下落延迟）, duration

### 3. 持续法术场 / 区域效果 (Area of Effect / Persistent Zone)
**描述**: 在指定位置创建一个持续存在的区域，对区域内的单位持续施加效果。
**游戏实例**: 毒液场、火焰之地、冰冻地面、神圣之地
**MagicWorld 适配**: 🆕 需新增
- 持续时间内 Tick 检测区域内目标
- 参数: targetPos, radius, duration, tickInterval, damagePerTick
- 可能需要新的 AZoneEffect Actor 或复用 ABullet + OverlapComponent

### 4. 光环 (Aura / Herald)
**描述**: 以施法者为中心的持续区域效果，跟随角色移动，对范围内友军/敌军施加增益/减益。
**游戏实例**: 生命光环、攻速光环、恐惧光环
**MagicWorld 适配**: 🆕 需新增
- 附着在角色身上，不是独立 Actor
- 参数: radius, buffType, value, duration（或永久 toggle）
- 可通过 skill_comp 管理激活/关闭

### 5. 召唤物 (Summon / Minion)
**描述**: 生成一个可控/自主的实体（宠物、傀儡、图腾），拥有独立 AI 和生命值。
**游戏实例**: 骷髅召唤、火元素、战斗图腾
**MagicWorld 适配**: 🆕 需新增
- 需要独立 AI 控制器
- 参数: summonClass, count, duration, aiBehavior
- 复杂度最高，需要完整的 AI 行为树

### 6. 自身增益 / Buff (Self-Buff / Party Buff)
**描述**: 对自身或队友施加临时增益效果（护盾、加速、无敌等）。
**游戏实例**: 魔法护盾、狂暴、神圣之光
**MagicWorld 适配**: 🆕 需新增
- 不生成外部 Actor
- 直接修改角色属性或添加 BuffComponent
- 参数: buffType, value, duration

### 7. 冲刺 / 位移 (Dash / Movement)
**描述**: 快速移动到目标位置，可能对路径上的敌人造成伤害。
**游戏实例**: 闪现、突刺、翻滚
**MagicWorld 适配**: 🆕 需新增
- 直接操控角色位置/速度
- 参数: distance, speed, damageOnPath, invulnerable

### 8. 吟唱 / 持续施法 (Channeling)
**描述**: 按住释放，持续产生效果，松手或被打断则停止。
**游戏实例**: 死亡射线、生命吸取、旋风斩
**MagicWorld 适配**: 🆕 需新增
- 需要输入状态管理（按住/松手）
- 参数: channelDuration, tickInterval, interruptible

### 9. 近战 / 直接打击 (Melee / Direct Hit)
**描述**: 近距离物理攻击，可能带范围扫击。
**游戏实例**: 斩击、重击、旋风斩
**MagicWorld 适配**: 🆕 需新增
- 播放攻击动画 + 碰撞检测框
- 参数: range, arc, hitCount, knockback

### 10. 陷阱 (Trap)
**描述**: 放置在地面，敌人接近时自动触发。
**游戏实例**: 火焰陷阱、冰冻陷阱、爆炸陷阱
**MagicWorld 适配**: 🆕 需新增
- 静态 Actor + OverlapSphere 检测触发
- 参数: triggerRadius, armTime, duration, effectOnTrigger

### 11. 地雷 (Mine)
**描述**: 放置在脚下，手动引爆。
**游戏实例**: 远程地雷、毒气地雷
**MagicWorld 适配**: 与陷阱类似，区别在于手动引爆而非自动触发

### 12. 图腾 (Totem / Turret)
**描述**: 放置静止实体，自动施放关联技能。
**游戏实例**: 火焰图腾、治疗图腾、弩炮
**MagicWorld 适配**: 🆕 需新增
- 类似召唤物但无移动 AI，固定位置自动攻击
- 参数: totemClass, attackSkill, duration, attackInterval

### 13. 烙印 (Brand / Mark)
**描述**: 附着在敌人身上，自动施放效果，目标死亡后转移到附近敌人。
**游戏实例**: 风暴烙印、末日烙印
**MagicWorld 适配**: 🆕 可后续考虑

### 14. 变形 (Transformation / Metamorphosis)
**描述**: 临时改变角色形态、属性和技能组。
**游戏实例**: 狼人变身、恶魔形态
**MagicWorld 适配**: 🆕 可后续考虑

## MagicWorld 优先级建议

| 优先级 | 类型 | 理由 |
|--------|------|------|
| P0（现有） | 投射物 Projectile | 已实现，ABullet 体系 |
| P1（推荐） | 落点法术 Ground Target | 落雷等核心法术，复用 ABullet 即可 |
| P1（推荐） | 持续法术场 Persistent Zone | 毒液场等，需新 Actor 但逻辑简单 |
| P2（中期） | 光环 Aura | 团队增益，需 BuffSystem |
| P2（中期） | 自身增益 Self-Buff | 护盾等，需 BuffSystem |
| P2（中期） | 近战 Melee | 扩展战斗风格 |
| P3（后期） | 召唤 Summon | AI 行为树复杂度高 |
| P3（后期） | 陷阱/地雷 Trap/Mine | 特殊玩法 |
| P3（后期） | 图腾 Totem | 类似召唤但简化 |
| P3（后期） | 吟唱 Channeling | 需输入系统支持 |
| P4（远期） | 冲刺/位移 Dash | 操控角色运动 |
| P4（远期） | 烙印/变形 Brand/Transform | 高级特殊玩法 |

## 与现有被动系统的关系

现有被动类型（`releaseType: passive`）是对主技能的修饰，不受形态类型影响：
- **弹道修饰**: StraightShot, CurveHoming, Accelerate（适用于 Projectile）
- **分裂修饰**: Split, RingSplit, Bounce, BounceBlast, BounceScatter（适用于 Projectile）
- **特效修饰**: FireExplosion, SkyDrop, SpiralTracking（可扩展到其他形态）

新增形态类型时，部分被动可能需要适配（如 Split 对 Ground Target 无意义，但对 Summon 可能有变体含义）。
