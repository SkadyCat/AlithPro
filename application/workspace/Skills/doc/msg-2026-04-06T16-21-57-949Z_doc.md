`msg-2026-04-06T16-21-57-949Z.md` — 基于 AIGC 生成技能图标

## 目标
基于之前的 ComfyUI AIGC 管线，创建可批量生成全部 600 个技能图标的脚本。

## 改动

### 新建 `AIGC/application/generate-catalog-icons.py`
从 `skill-catalog.json` 读取 600 个技能，自动生成 ComfyUI prompt 并提交生成。

**核心特性**:
1. **Prompt 增强**: 根据技能特征自动组合视觉关键词
   - 主技能: 元素色彩 + 形态描述 + 类名
   - 辅助石: 分类视觉 + 节点属性线索（elementType, controlType, trajectoryType）+ 中文名
   - 统一后缀: `rpgicondiff` trigger + `hand painted style, dark background, game UI icon`
2. **视觉关键词映射**: 10 种分类 + 10 种形态 + 10 种元素各有独特描述
3. **断点续传**: `--skip N` 跳过前 N 个，`--limit N` 限制数量
4. **过滤**: `--type primary/passive` + `--cat split/trajectory/...`
5. **预览**: `--dry-run` 显示 prompt 不执行
6. **清单**: 生成 `catalog_icons_manifest.json`

**ComfyUI 参数** (与原脚本一致):
- 模型: `handpaintedRPGIcons_v1.safetensors`
- 512×512, 25 steps, euler_ancestral, cfg 7.5
- 种子: 基于 assetId 取模确保可重现

**用法示例**:
```bash
# 全量生成（约 600 张，预计 10+ 小时）
python generate-catalog-icons.py --catalog path/to/skill-catalog.json

# 只生成辅助石的分裂类（50 张）
python generate-catalog-icons.py --catalog ... --type passive --cat split

# 断点续传：从第 201 张开始
python generate-catalog-icons.py --catalog ... --skip 200

# 预览 prompt
python generate-catalog-icons.py --catalog ... --dry-run
```

## 验证
- ✅ 全量 dry-run: 600 tasks 正确加载
- ✅ 分类过滤: `--type passive --cat split` → 50 tasks
- ✅ skip/limit: `--skip 595 --limit 5` → 最后 5 个
- ✅ Prompt 质量: 含 rpgicondiff trigger + 分类视觉 + 元素色 + 中文名
