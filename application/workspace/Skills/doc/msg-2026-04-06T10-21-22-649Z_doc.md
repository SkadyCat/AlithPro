# 主动技能无被动时投射物为何移动

**源任务**: `msg-2026-04-06T10-21-22-649Z.md`
**目标**: 修复主动技能无辅助被动修饰时投射物不应移动的问题

## 根因

`skill_comp.lua` 第 273-278 行有一段 **速度回退逻辑**：

```lua
if vel.X == 0 and vel.Y == 0 and vel.Z == 0 then
    local speed = primary.speed or primary.Speed or 1000
    bullet.ProjectileMovement.Velocity = dir * speed
end
```

这是之前为了保证"至少能飞"而加的安全回退。但在 AS 原版设计中：
- 主技能本身不设速度，只负责生成子弹
- 移动行为由被动石决定（直射 StraightShot、曲线追踪 CurveHoming 等）
- 无被动 = 子弹原地不动

## 修复

删除速度回退逻辑，改为仅打印日志。无被动修饰时子弹保持零速度（与 AS 原版行为一致）。

## 改动文件

- `Content/Script/com/components/skill_comp.lua` — 移除 dir*speed 回退，保持零速度

## 验证

- ✅ 代码逻辑与 AS 原版 SkillManager.DoSkill() 行为一致
- ⚠️ 需要 UE 重启验证：装备主技能但不装被动时，子弹应原地不动
