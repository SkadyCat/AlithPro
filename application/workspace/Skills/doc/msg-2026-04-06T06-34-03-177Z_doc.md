# 交付说明：显示 DA 资产实例

## 源任务
`msg-2026-04-06T06-34-03-177Z.md` — 要求展示 Content/DA 下的实际 Asset 实例，而非仅类定义。

## 目标
在技能节点可视化工具中新增"实例视图"，扫描 MagicWorld Content/DA/ 目录下的 .uasset 文件，展示每个 DA 资产实例（如 DA_Bounce、DA_PrimaryAbility_FireBall 等），并支持与类定义视图之间的交叉导航。

## 改动内容

### server.js
- 新增 `CONTENT_DA_ROOT` 和 `EXPORTED_JSON` 路径常量
- 新增 `DA_NAME_CLASS_MAP` 文件名到类名的映射表
- 新增 `scanDAInstances()` 函数：遍历 Content/DA/Ability/ 和 Content/DA/Primary/ 目录，收集所有 .uasset 文件作为实例
- 从文件名推断所属类（如 DA_PrimaryAbility_FireBall → UMagicPrimary）
- 支持可选的 ExportedAssets.json 覆盖（UE 编辑器导出后可展示实际属性值）
- `/api/skills` 响应新增 `instances` 数组

### app.js
- 完全重写为双视图模式（实例视图 / 类定义视图）
- 新增 `renderInstanceMenu()` — 按分类渲染实例列表
- 新增 `renderInstanceDetail()` — 展示实例详情（Content 路径、推断类、类属性、导出值）
- 新增 `switchToClassAndSelect()` — 从实例跳转到对应类定义
- 视图切换按钮联动菜单和详情区

### index.html
- 菜单头部新增"类定义 / 实例"切换按钮 `btn-toggle-view`

### style.css
- 新增 `.tag-instance` 样式（紫色标签）
- 重构按钮样式，统一 `btn-refresh` 和 `btn-toggle-view`

## 验证结果
- `node --check server.js` — 语法正确 ✓
- API 返回 153 class nodes + 13 DA instances ✓
- HTML 页面包含 toggle 按钮 ✓
- 服务运行在 http://localhost:3220 ✓

## 已知限制
- .uasset 为二进制格式，无法直接读取属性值；需 UE 编辑器运行 `ExportAllSubAssetsToJson()` 后才能获取实际属性
- 类推断基于文件名模式匹配，新增 DA 文件名不匹配已知模式时会回退到 "Unknown"

## 风险
无阻塞风险。功能完整可用。
