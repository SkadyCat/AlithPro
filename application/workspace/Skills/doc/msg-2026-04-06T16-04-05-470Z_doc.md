# 交付说明 — 技能石详情展示修复

## 源任务
`msg-2026-04-06T16-04-05-470Z.md` — projectile-design.html 无法查看具体技能石信息

## 目标
辅助石列表全部折叠且没有展开提示，用户无法看到具体的技能石参数信息。

## 问题分析
1. 辅助石分类卡片（30-80个技能）默认全部折叠（`display:none`），无展开指示
2. 分类头部无视觉提示（如箭头），用户不知道可以点击展开
3. 展开后仅显示名称+描述+小标签，缺少查看每个技能石完整属性的方式

## 改动

### projectile-design.html
1. **分类卡片折叠指示器**: 添加 `▶` 箭头到分类头部，展开时旋转为 `▶`（CSS transform），文字提示"点击展开/收起"
2. **技能石可点击展开详情**: 每行右侧显示"▼ 详情"按钮，点击展开详情面板
3. **详情面板包含**:
   - 📋 基本信息：ID、类名、分类、形态、元素、行为
   - ⚙️ 节点属性：分类特定的游戏参数（如 trajectoryType, splitCount, damageBonus 等）
   - 🎨 AIGC 提示词：图标生成用的 prompt
4. **交互改进**: skill-row hover 背景高亮，cat-body 高度上限从600→800px
5. **CSS新增**: `.skill-detail` / `.detail-grid` / `.detail-item` / `.toggle-arrow` 样式

## 验证
- ✅ 页面正常加载 (200, 29KB)
- ✅ toggle-arrow、skill-detail、nodeProps 逻辑全部存在
- ✅ 辅助石展开后可逐个点击查看完整节点属性
