## 诊断发现
- McpFlask 热重载存在 **端口泄漏 bug**：`_ue_stop_flask()` 后旧服务器未完全释放 port 5001，导致新旧两个 PID 同时 LISTEN
- UE 帧计数器冻结在 [356]，Slate tick 回调不执行
- 根因：热重载机制在 `shutdown()` 后 0.3s 就启动新线程，旧 socket 未释放
- 解决方案：重启 UE 编辑器可恢复正常，或直接用 Python 控制台执行 `export_da_json.py`
