## 目标
在检查器模式下，右键控件弹出菜单，可编写修改意见、保存到本地、一键汇总提交到 UITS 工作队列。

## 改动

### server.js — 3 个新 API 端点
- `POST /api/save-suggestion`：接收 `{controlId, markdown}`，写入 `public/suggestions/{controlId}_{timestamp}.md`。
- `GET /api/list-suggestions`：列出 suggestions 文件夹中所有 .md 文件及内容。
- `POST /api/submit-suggestions`：汇总所有意见为一份文档，写入 `sessions/suggestions-{timestamp}.md`，清空 suggestions 文件夹。

### bag-preview-umg.html — UI 交互
- **右键菜单**：检查器开启状态下，右键控件弹出 `.ctx-menu`，含"📝 修改意见"按钮。
- **弹窗编辑器**：`.modal-overlay` 含控件信息显示、Markdown textarea、三个操作按钮（取消/保存/提交到 UITS）。
- **意见计数徽章**：提交按钮旁显示当前已保存意见数量。
- **Toast 通知**：保存/提交结果以底部浮窗反馈。

### 新建目录
- `application/public/suggestions/` — 修改意见本地存储。
