## 目标
提供修改意见的卡片列表查看器，支持右键编辑和删除操作。

## 改动

### server.js — 2 个新 API
- `POST /api/delete-suggestion`：删除指定意见文件。
- `POST /api/update-suggestion`：更新指定意见文件内容。

### bag-preview-umg.html — UI
- **"📋 查看全部意见"按钮**：在修改意见弹窗 footer 中新增，点击打开卡片列表。
- **卡片列表弹窗** `.cards-overlay`：列出所有已保存意见，每张卡片显示文件名和内容预览。
- **卡片右键菜单** `.card-ctx-menu`：
  - ✏️ 编辑：关闭卡片列表，将内容加载到 Markdown 编辑器中，保存时调用 update API。
  - 🗑️ 删除：调用 delete API，刷新列表和计数。
- **保存按钮增强**：同一个保存按钮自动识别新建/编辑模式（通过 `dataset.editFilename`），编辑模式调用 update API。

### 新增 CSS
- 卡片列表弹窗、卡片样式、卡片右键菜单样式。
