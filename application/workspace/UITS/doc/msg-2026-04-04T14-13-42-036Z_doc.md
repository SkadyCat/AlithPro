## 改动

### elements/UTileView.js — 全面增强
**新增属性（observedAttributes）**：
- `entry-width` / `entry-height`：UE5 EntryWidth/EntryHeight，控制 tile 尺寸
- `selection-mode`：Single / Multi / None，选中模式
- `orientation`：Vertical / Horizontal，排列方向
- `scrollbar-visibility`：Auto / AlwaysOn / AlwaysOff
- `tile-alignment`：start / center / end，对齐方式
- `enable-scroll-animation`：滚动动画开关

**新增 JS 属性 getter**：
- `EntryWidth` / `EntryHeight` / `Columns` / `Gap` / `SelectionMode` / `Orientation` / `ScrollbarVisibility` / `TileAlignment` / `NumItems`

**新增公开方法**：
- `getSelectedIndex()` / `getSelectedIndices()` / `getSelectedItems()`
- `setSelectedIndex(idx)` / `clearSelection()`
- `scrollToItem(idx)`
- `getNumItems()`

**新增事件**：
- `OnItemDoubleClicked`：双击 tile 触发
- `OnSelectionChanged`：选中状态变化时触发

**增强行为**：
- Multi 选中模式下 toggle 选中
- Horizontal 方向支持（grid-auto-flow: column）
- `attributeChangedCallback` 支持动态属性更新重新渲染
