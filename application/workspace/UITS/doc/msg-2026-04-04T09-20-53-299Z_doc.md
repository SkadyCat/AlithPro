# 交付说明 — UMG 控件库 (HTML Web Components)

## 源任务
`msg-2026-04-04T09-20-53-299Z.md` — "基于这些控件的特性，我们要维护(封装) 一份功能完全一样，名字完全一样的 控件库 到 elements 文件夹。"

## 目标
基于上一轮生成的 UMG 控件参考文档，在 `application/elements/` 中创建一套与 UE5 UMG 同名同功能的 HTML Web Components 控件库。

## 改动

### 新增文件（8 个 JS + 1 个 README）
- **elements/UWidget.js** — 基类，提供 Visibility/RenderOpacity/IsEnabled/ToolTipText/Padding 通用属性
- **elements/UButton.js** — Button，四态样式（Normal/Hovered/Pressed/Disabled），OnClicked 等 5 个事件
- **elements/UTextBlock.js** — TextBlock（字体/颜色/对齐/换行/阴影/描边）+ RichTextBlock
- **elements/UImage.js** — Image/ProgressBar/Throbber/CircularThrobber/BackgroundBlur（5 个控件）
- **elements/UInputControls.js** — CheckBox/EditableText/EditableTextBox/MultiLine 两种/SpinBox/Slider/ComboBoxString（8 个控件）
- **elements/UContainers.js** — CanvasPanel/HBox/VBox/WrapBox/Grid/UniformGrid/Overlay/WidgetSwitcher/ScrollBox/SizeBox/ScaleBox/Spacer/Border/SafeZone（14 个控件）
- **elements/UDataViews.js** — ListView/TileView/TreeView/MenuAnchor（4 个控件）
- **elements/index.js** — 入口文件，导入即注册全部 34 个自定义元素
- **elements/README.md** — 中文说明，含控件清单、属性表、文件结构、使用示例

### 服务器接入
- 创建 `public/elements` → `../elements` 的 Junction，使控件库可通过 `http://127.0.0.1:3200/elements/` 访问
