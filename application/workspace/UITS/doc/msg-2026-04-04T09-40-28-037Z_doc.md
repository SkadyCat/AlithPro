## 使用的 UMG 控件
| 节点 | 控件 | 用途 |
|------|------|------|
| RootCanvas | `<umg-canvas-panel>` | 1440×900 绝对定位根容器 |
| Background | `<umg-border>` | 纯色背景填充 |
| CapacityPlate | `<umg-border>` | 容量牌背景（渐变 + 圆角） |
| CapacityLabel | `<umg-text-block>` ×3 | 已用/分隔符/总计 |
| CloseButton | `<umg-button>` | 关闭按钮（四态样式） |
| Tabs | `<umg-vertical-box>` | 标签栏容器 |
| Tab×4 | `<umg-button>` | 装备/道具/材料/通货标签 |
| InventoryPanel | `<umg-overlay>` | 层叠面板 |
| GridContainer | `<umg-uniform-grid-panel>` | 3×4 物品网格 |
| NodeTree | `<umg-tree-view>` | 右侧节点树 |
| ScaleButtons | `<umg-button>` ×4 | 缩放控制 |

- 关闭按钮动画
- 缩放控制（50/65/80/100%）
