任务目标：
- 探索并落地一个页面：使用 CSS 生成固定分辨率的按钮背景图，再在 HTML 的 canvas 中绘制出来，并提供可直接打开的网址。

所做改动：
- 新增 `application\public\button-canvas.html`：
  - 左侧展示 CSS 生成的固定分辨率按钮背景；
  - 右侧展示绘制到 `<canvas>` 后的结果。
- 新增 `application\public\button-canvas.css`：
  - 定义固定尺寸 `320 x 96` 的按钮背景；
  - 用渐变、描边、内阴影、高光和底部铭文条构成按钮视觉。
- 新增 `application\public\button-canvas.js`：
  - 读取页面 CSS；
  - 使用 SVG `foreignObject` 将 CSS 按钮栅格化；
  - 把结果绘制进 `canvas`；
  - 在页面中显示当前绘制状态。

可打开地址：
- `http://127.0.0.1:3200/button-canvas.html`

验证结果：
- `npm run check` 通过。
- 访问 `http://127.0.0.1:3200/button-canvas.html` 返回 200。
- 页面内容已确认包含 `button-bg` 与 `render-canvas`，说明按钮源和 canvas 结果区域都已就位。

剩余风险或阻塞项：
- 当前 canvas 绘制依赖浏览器对 SVG `foreignObject` 的支持；在现代 Chromium 环境下可用，但如果后续要兼容更广范围环境，可能需要改成 DOM 截图库或纯 canvas 绘制实现。
