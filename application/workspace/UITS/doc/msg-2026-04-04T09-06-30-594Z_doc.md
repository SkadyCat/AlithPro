# 交付说明：HTML 背包 UI 预览

## 源任务文件
`msg-2026-04-04T09-06-30-594Z.md`

## 目标
基于 bag.bp 的四区结构和 bag_dark_layout.json 坐标，用 HTML/CSS/JS 实现可交互的背包 UI 预览页面。

## 改动内容

### 新增文件
- `application\public\bag-preview.html` — 单文件 HTML 预览，无外部依赖。

### 功能特性
- **四区布局**：容量栏 (302,113)、关闭按钮 (640,140)、左侧页签 (173,220)、中部格子 (300,160)，坐标与 bag_dark_layout.json 一致。
- **暗黑风格**：#070810 底色、冷蓝灰配色、渐变边框，对齐 bag.bp 的视觉方向。
- **分类切换**：点击左侧页签（装备/道具/材料/通货）切换物品列表。
- **物品选中**：点击格子选中物品，高亮显示。
- **Tooltip**：悬停显示物品详情（名称、属性、说明）。
- **缩放控件**：50% / 65% / 80% / 100%，默认 80%。
- **数据**：8 个示例物品与 main.gd 中 ITEMS 一致。

### 访问方式
- 直接打开文件：`application\public\bag-preview.html`
- 通过服务器：`http://127.0.0.1:3200/bag-preview.html`（需先启动 server.js）

## 验证结果
- `npm run check` 通过。
- 文件为自包含 HTML，浏览器直接打开即可预览。

## 风险与阻塞
- 无。HTML 预览为独立文件，不影响现有管线。
