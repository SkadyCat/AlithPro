# 交付说明

- 源任务文件：`msg-2026-04-02T17-21-28-238Z.md`
- 任务目标：解决背包没有图片的问题，改为创建一个 Node.js 服务，根据控件结构构造 CSS 效果并导出图片。

## 所做改动

- 在现有 `application\server.js` 中新增 `GET /api/export-image` 导出接口。
- 引入 `playwright`，由 Node.js 服务端加载专用 CSS 背包页面并截图生成 PNG。
- 新增专用导图页面与资源：
  - `application\public\inventory-export.html`
  - `application\public\inventory-export.css`
  - `application\public\inventory-export.js`
- 新增 `application\public\exports\` 目录用于保存导出的图片。
- 在原工作台页中新增：
  - 顶部按钮 `导出当前背包图`
  - 右侧导出状态区
  - 导出图片链接与预览图
- 修正了导图验证阶段的服务冲突问题：之前 3200 端口被旧 Node 进程占用，导致静态页是新文件，但服务路由仍是旧逻辑；现已切换为最新服务实例。

- 当前导图能力优先覆盖背包类页面（默认 / 国风 / 科幻 / 轻日式），其他场景如 HUD / 对话暂未接入导图模板。
- 导出的图片仍基于 CSS 形状与文本构成，不包含外部美术贴图；如果后续要做更接近商业成图，可继续补贴图资源和图标素材。
