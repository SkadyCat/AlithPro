任务目标：
- 制作一个网页：
  - 可以绘制边框；
  - 可以选中边框；
  - 选中后可以描述这个边框是什么样的；
  - 最终把这种基础布局用一个 JSON 描述出来，便于后续传给我继续生成背包布局。

所做改动：
- 新增页面：
  - `application\public\layout-annotator.html`
  - `application\public\layout-annotator.css`
  - `application\public\layout-annotator.js`
- 页面能力：
  - 在画布空白区域拖拽创建矩形框；
  - 点击矩形框进行选中；
  - 为选中框填写：
    - `id`
    - `kind`
    - `label`
    - `description`
  - 右侧实时生成 JSON；
  - 支持复制 JSON；
  - 支持下载 `layout-annotation.json`。
- JSON 结构：
  - `schema`
  - `canvas.width`
  - `canvas.height`
  - `regions[]`
    - `id`
    - `kind`
    - `label`
    - `description`
    - `x`
    - `y`
    - `width`
    - `height`
- 额外修正：
  - 编辑框 ID 时，同步更新当前选中状态，避免编辑后丢失选中框。

访问地址：
- `http://127.0.0.1:3200/layout-annotator.html`

验证结果：
- `npm run check` 通过。
- 页面、JS、CSS 地址返回 200：
  - `/layout-annotator.html`
  - `/layout-annotator.js`
  - `/layout-annotator.css`
- 已确认脚本中包含导出 schema：`layout-annotation.v1`

剩余风险或阻塞项：
- 当前是基础布局标注工具，支持框级描述；如果后续需要更复杂的层级关系、吸附、缩放、背景图导入或分组，还需要继续扩展。
