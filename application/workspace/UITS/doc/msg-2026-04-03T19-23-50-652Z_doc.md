## 调研结论
- **可以做相近效果，但是否能纯 CSS 复刻，取决于图片类型。**
- 适合纯 CSS 的情况：
  - 几何面板、边框、描边、内外发光、渐变、高光、凹凸压印、玻璃感、金属感
  - 规则图形、装饰条、角标、按钮、卡片、HUD 面板
- 不适合纯 CSS 还原的情况：
  - 手绘插画
  - 高频纹理
  - 复杂噪声
  - 非规则笔触
  - 细碎材质细节
- 这类复杂视觉通常应采用：
  - **CSS 负责结构、光影、交互态**
  - **切图/贴图负责复杂纹理与装饰细节**

## 外网调研要点
- `mask-image`
  - 可用渐变或图片作为遮罩，适合做非矩形裁切、镂空、边角造型
- `filter`
  - 可做 `blur`、`brightness`、`contrast`、`drop-shadow` 等效果，适合发光、阴影、整体色调修正
- `box-shadow`
  - 支持多层阴影和 `inset`，适合做浮雕、内阴影、边缘厚度和层次感
- `background-blend-mode`
  - 可叠加多层背景并指定混合模式，适合做金属、染色、压暗、叠光等视觉

## 方案建议
1. 先把目标图拆成 3 层：
   - **结构层**：尺寸、圆角、边框、分区、留白
   - **光影层**：渐变、高光、内阴影、外发光、滤镜
   - **纹理层**：噪声、花纹、复杂装饰、图案
2. 对结构层和光影层优先使用纯 CSS：
   - `linear-gradient / radial-gradient`
   - `box-shadow / inset box-shadow`
   - `filter / drop-shadow`
   - `background-blend-mode`
   - `mask-image`
   - `::before / ::after`
3. 对纹理层采用小图切片或透明 PNG/SVG：
   - 角花
   - 金属纹理
   - 特殊磨损
   - 非规则雕刻
4. 最终形成混合方案：
   - **80% 视觉骨架用 CSS**
   - **20% 难以复刻的材质细节用贴图**
