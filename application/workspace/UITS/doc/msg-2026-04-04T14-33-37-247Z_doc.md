## 所做改动

### 新建文件
1. **`application/public/panels/bag-panel.js`** — 背包面板模块（ES module），导出 name/icon/id/css/html/treeNodes/treeLabelMap/tabCatMap/init 函数
2. **`application/public/editor.html`** — 全新 5 区域编辑器（~550 行，36KB）

### editor.html 功能概览
- **top_menu**: 面板标题、检查器开关、缩放控制（50%/65%/80%/100%）
- **left_menu**: 面板列表侧栏（点击加载不同面板到 center_area）
- **center_area**: 动态面板渲染区（从 panels/ 加载 HTML + CSS + init）
- **center_bottom**: 内联修改意见卡片区（带提交按钮）
- **right_info**: 节点树 + 属性面板
- 完整功能：检查器（悬停/高亮/属性面板含 TileView/ListView 专区）、树视图双向同步、右键菜单→意见编辑器、Markdown 预览、建议 CRUD + 提交到 UITS、Toast 通知

### 架构设计
- 面板模块化：panels/ 下的 ES module 通过 `import()` 动态加载
- 共享系统（inspector、context menu、suggestions、MD preview）全部内置在 editor.html
- 面板特定 CSS 通过 `<style id="panelStyle">` 注入，切换面板时替换
- 未来扩展只需在 panels/ 添加新模块文件
