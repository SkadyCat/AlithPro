# 交付说明: 词条来源弹窗（hover + 点击固化）

## 源任务
`msg-2026-04-06T13-46-07-764Z.md`

## 目标
计算链中的贡献标签 (contrib-badge) hover 显示来源道具弹窗，点击后弹窗固化并带关闭按钮。

## 改动

### `app.js` (前端)
- `contrib-badge` 增加 `data-slot`、`data-item`、`data-mod-text` 属性
- 新增 `initContribPopups()`: 为每个 badge 绑定 mouseenter/mouseleave/click 事件
- 新增 `showItemPopup(badge, pinned)`:
  - 从 `allMods` 筛选同槽位所有词条
  - 高亮触发词条 (`.popup-mod-highlight`)
  - 禁用词条半透+删除线 (`.popup-mod-disabled`)
  - `pinned=true` 时显示关闭按钮 (✕) 和蓝色边框
- 新增 `closePopup()`: 关闭固化弹窗
- 全局点击监听: 点击弹窗外部自动关闭固化弹窗
- hover 200ms 延迟防止闪烁

### `style.css`
- `.item-popup`: 固定定位、暗色背景、圆角、阴影
- `.item-popup.pinned`: 蓝色边框+发光
- `.item-popup-header`: 道具名 + 槽位 + 关闭按钮
- `.popup-mod-highlight`: 蓝色高亮当前触发词条
- `.popup-mod-disabled`: 半透删除线
