## 改动

### `application/public/js/app.js`
1. **重构弹窗系统**:
   - 提取 `createItemPopupEl(badge, pinned)` — 创建装备词条弹窗 DOM 元素
   - 提取 `positionPopup(popup, anchorRect)` — 通用弹窗定位
   - `showItemPopup()` 改为调用上述两个辅助函数
2. **新增子弹窗跟踪**: `_subPopup` 和 `_subHoverTimer` 变量
3. **分解弹窗词条行为**:
   - **hover (200ms)**: 在父弹窗右侧显示装备词条弹窗（非固定）
   - **mouseleave (150ms)**: 延迟关闭，允许光标移至子弹窗
   - **click**: 固定子弹窗（pinned 样式），父弹窗保持不动
   - 子弹窗自带 mouseenter/mouseleave 防止光标在上面时被关闭
4. **关闭逻辑**:
   - `closePopup()` 同时处理 `_subPopup`
   - outside-click: 先关子弹窗，再关父弹窗（级联关闭）
