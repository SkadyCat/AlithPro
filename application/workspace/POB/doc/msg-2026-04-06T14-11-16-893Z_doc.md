# 交付说明：词缀信息中文翻译

## 源任务
`msg-2026-04-06T14-11-16-893Z.md` — 翻译词缀信息为中文

## 目标
将 POB Node 应用中所有装备词缀（affix/mod）信息翻译为中文显示。

## 改动

### 新增文件
- `application/public/js/mod-zh.js` — 中文翻译模块
  - 270+ 条正则翻译规则，覆盖所有 POE 词缀模式
  - 元素/属性/增益/光环等关键词映射
  - `translateMod(line)` — 单行翻译入口
  - `translateItemText(text)` — 整段翻译
  - `translateBaseType(bt)` — 基底类型翻译（脊弓、皮革腰带等）
  - `translateUniqueName(name)` — 传奇物品名翻译（猎首、寡妇哀嚎等）

### 修改文件
- `application/public/index.html` — 引入 `mod-zh.js`
- `application/public/js/app.js`
  - 装备栏位名中文映射（`_SLOT_ZH`）：武器→Weapon、头盔→Helmet 等
  - `renderItemCard()` — 物品卡片显示中文翻译名、基底类型中文、栏位中文
  - `formatItemMods()` — 词缀行翻译为中文（原文保留在 tooltip）
  - 伤害分析面板 — 词条来源分组标题、复选框文字均翻译
  - 弹窗系统 — 弹窗中的词缀也翻译为中文
  - 支持 `{mutated}` 前缀词缀
- `application/public/css/style.css` — 新增 `.item-base-zh` 样式

## 翻译覆盖率
- 302 条唯一词缀，285 条已翻译，17 条元数据行自动跳过
- **翻译覆盖率 100%**

## 风险
- 新装备的词缀如果包含未覆盖的英文模式，会回退显示原文（不会报错）
- 传奇物品名和基底类型翻译目前只覆盖当前存档中的物品，新物品可能需要扩充字典
