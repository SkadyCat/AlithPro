# 交付说明 — 分析每件装备绑定的技能石

## 源任务
`msg-2026-04-06T08-59-48-246Z.md`

## 目标
分析 POB 构建中每件装备绑定了哪些技能石（Gem），输出装备→技能石的完整映射。

## 实现内容

### 新建文件
- **`application/src/analysis/gem-setup.js`** — 装备-技能石分析模块
  - `analyzeGemSetup(build)` 函数，接收解码后的 Build 对象
  - 解析 `<Skill slot="...">` 与 `<Slot name="..." itemId="...">` 的关联关系
  - 支持 12 个标准装备槽位（主副手、头胸手鞋、项链戒指腰带）及自定义槽位
  - 中文槽位标签（主手/副手/头盔/胸甲/手套/鞋子…）
  - 从物品文本中解析装备名称和基底类型
  - 统计：总宝石数、活跃槽位数、主技能所在槽位

### 修改文件
- **`application/src/routes/build.js`** — 新增 `POST /api/build/gem-setup` 端点
- **`application/src/core/build-codec.js`** — 修复 `<ItemSet>` 下的 `<Slot>` 解析
  - `ItemSet` 加入 `isArray` 列表
  - `normalizeItems()` 现在从活跃 `ItemSet` 中提取槽位映射

## API 使用
```
POST /api/build/gem-setup
Body: { "code": "<POB构建代码>" }
```
