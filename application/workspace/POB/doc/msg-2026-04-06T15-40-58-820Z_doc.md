# 交付说明: 分解弹窗词条可交互

## 源任务
`msg-2026-04-06T15-40-58-820Z.md` — 装备增伤弹窗内的词条也要可交互，显示对应装备的哪条词条

## 目标
分解弹窗中的每个装备/宝石条目可点击，点击后打开该装备的完整词条弹窗，并高亮对应的词条。

## 改动

### `application/src/analysis/damage-breakdown.js`
- 分解 items 新增 `source` (栏位如 "Flask 4") 和 `modText` (原始词条文本) 字段

### `application/public/js/app.js`
- `showDecompPopup()`: 有 source 的条目加 `.decomp-item-link` 类和 `data-slot/data-item/data-mod-text` 属性
- 点击 `.decomp-item-link` 行: 关闭当前弹窗 → 调用 `showItemPopup(row, true)` 打开装备词条弹窗
- 复用现有 showItemPopup 逻辑: 查找该 slot 所有词条，高亮匹配行

### `application/public/css/style.css`
- `.decomp-item-link`: cursor pointer、hover 蓝色背景
- `.decomp-item-link .decomp-item-name`: 点线下划线提示可交互
