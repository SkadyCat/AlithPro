# 交付说明: 计算链直接显示调整后的值

## 源任务
`msg-2026-04-06T13-27-55-395Z.md`

## 目标
用户要求取消词条后，计算链的数值应该直接显示调整后的结果，不需要 "➜ 估算" 之类的标注。

## 改动

### `damage-breakdown.js` (后端)
- 当 `disabledSet.size > 0` 时:
  - **平均单次伤害**: 直接显示 `estAvgDmg`（非原始 `avgDmg`）
  - **最终攻速**: 直接显示 `estSpeed`（非原始 `speed`）
  - **总 DPS**: 直接显示 `estDPS`（非原始 `totalDPS`）
  - `detail` 中附带原始值作为参考（如 `(原始 8760691)`）
- 无词条禁用时仍显示 POB 原始值

### `app.js` (前端)
- 移除了 "chain-step-changed" 高亮和 "➜ 估算" 内联注释
- 因为值已直接替换，无需额外 UI 标注
