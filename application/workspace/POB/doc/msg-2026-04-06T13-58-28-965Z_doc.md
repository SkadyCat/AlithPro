# 交付说明: 最终攻速计算分解

## 源任务
`msg-2026-04-06T13-58-28-965Z.md`

## 目标
用户问 "最终攻速怎么算出来的？"，需要在计算链中展示完整的攻速推导过程。

## 改动

### `damage-breakdown.js`
- "最终攻速" 链步骤增加完整计算分解:
  - 反向推算: `otherMult = finalSpeed / (weaponPanel × gearGlobalMult)`
  - detail 格式: `基底 X × (1+局部%) = Y × (1+全局%) = Z × W(天赋/宝石/充能) = 最终`
  - explanation: `= 武器基底 × (1+局部攻速%) × (1+装备全局攻速%) × 其他(天赋/宝石/充能等)`
- 禁用攻速词条时: 使用调整后的 weapon.attackRate 和 adjGlobalSpeedInc，末尾附 `(原始 X.XX)`

## 验证
| 场景 | 分解 |
|------|------|
| 无禁用 | 基底 1.4 × (1+27%) = 1.78 × (1+22%) = 2.17 × 3.06 = 6.65 ✅ |
| 禁用 8% AS | 基底 1.4 × (1+19%) = 1.67 × (1+22%) = 2.04 × 3.06 = 6.23 (原始 6.65) ✅ |

## 剩余风险
- "其他" 倍率 (3.06x) 是反推值，包含天赋树、技能宝石、充能球、光环等所有无法逐项分解的来源
