### 已实现功能
1. **POB Build Code 导入** — 完整 Base64 → zlib inflate → XML 解析管线
2. **Build 数据解析** — 角色信息、装备、技能组（Skills/SkillSet/Skill）、天赋树、配置
3. **PlayerStat 导入** — 读取 POB 预计算的 DPS/暴击/速度等数据
4. **ModDB 词缀系统** — 完整的 BASE/INC/MORE/FLAG/OVERRIDE/LIST 查询引擎
5. **防御计算** — 生命、魔力、ES、护甲、闪避、抗性（含上限）、格挡、压制
6. **攻击计算** — 使用 POB 导入数据或本地计算（含元素分类 DPS）
7. **Web UI** — 暗色主题、标签式浏览、Build 导入界面
8. **Build Code 导出** — XML 生成 → zlib deflate → Base64 编码

## 剩余风险或阻塞项
- **本地计算精度有限**：完整 DPS 计算需要移植 POB 全部游戏数据层（Gems.lua 396KB、CalcOffence.lua 340KB、CalcDefence.lua 209KB），当前优先使用 POB 导入的预计算数据
- **天赋树 Mod 解析**：需要加载 TreeData 并解析节点 → 词缀映射
- **装备词缀解析**：需要 ModParser 移植（词缀文本 → 结构化 Mod）
- **后续可扩展**：URL 导入（pastebin/pobb.in）、TradeQuery API、多 Build 对比
