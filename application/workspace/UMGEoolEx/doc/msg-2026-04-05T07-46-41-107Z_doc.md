# 交付说明 - UMG Blueprint Explorer

## 源任务
`msg-2026-04-05T07-46-41-107Z.md`

## 目标
创建 Node.js 应用，读取 `G:\GameExPro3\MagicWorld\Content\UI` 中所有 UMG 蓝图，左侧菜单显示文件树，选中蓝图后在中心区域以节点树形式紧凑渲染导出的 JSON 数据。

## 改动内容

### 新增文件（`application/` 目录下）
| 文件 | 说明 |
|------|------|
| `package.json` | 项目配置，依赖 express |
| `server.js` | Express 服务器，端口 3210，提供 `/api/tree` 和 `/api/parse` 接口 |
| `lib/scanner.js` | 递归扫描目录，生成 `.uasset` 文件树结构 |
| `lib/uasset-parser.js` | UE5 `.uasset` 二进制解析器 — 解析文件头、名称表、导入表（40字节/条）、导出表（112字节/条，含 TemplateIndex） |
| `public/index.html` | 主页面，三栏布局 |
| `public/style.css` | 暗色主题样式 |
| `public/app.js` | 前端逻辑：文件树渲染、搜索过滤、节点树渲染、JSON 导出、详情面板 |

### 功能特性
- **左侧菜单** (left_menu)：展示 `Content\UI` 下全部 652 个 `.uasset` 文件，支持折叠目录和实时搜索过滤
- **中心区域** (center_area)：选中蓝图后自动解析 `.uasset` 二进制格式，提取 WidgetTree 子树，以紧凑节点树渲染（显示类名、对象名、序列化大小）
- **右侧详情**：点击节点显示 export index、outer index、serial size 等详细信息
- **JSON 导出**：一键将节点树导出为格式化 JSON 文件
- **二进制解析器**：支持 UE4 v522 / UE5 v1012 格式，正确处理 FName(8字节)、import/export 表固定大小条目

## 验证结果
- `npm run check` 通过
- 服务器 `http://localhost:3210` 正常启动
- `/api/tree` 返回 15 个顶级目录/文件
- `/api/parse` 在 `WBP_ExamplePanel`（29 exports）和 `WBP_BagPanel`（147 exports）上均成功解析并输出完整 widget tree

## 剩余风险
- 解析器基于 UE5 v1012 二进制格式经验推导；其他 UE 版本的 `.uasset` 可能需要调整条目大小
- 非 Widget Blueprint 的 `.uasset`（纹理、数据资产等）不含 WidgetTree，此时回退显示完整 export 树
- 当前解析器未读取属性序列化数据（仅提取结构层级信息）
