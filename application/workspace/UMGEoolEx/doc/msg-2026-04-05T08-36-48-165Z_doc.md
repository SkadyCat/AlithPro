# 交付说明 — 右键修改意见功能

## 源任务
`msg-2026-04-05T08-36-48-165Z.md`

## 目标
在 UMG Explorer 中为树节点添加右键菜单，支持「修改意见」编辑、保存与投递到 sessions/ 队列。

## 改动

### server.js
- 引入 `fs`，新增 `DRAFTS_DIR` / `SESSIONS_DIR` 常量
- 添加 `express.json()` 中间件
- `GET /api/suggestions` — 列出所有草稿
- `POST /api/suggestions` — 保存建议到 `drafts/` JSON
- `DELETE /api/suggestions/:id` — 删除单条草稿
- `POST /api/submit` — 将所有草稿合并为 Markdown 写入 `sessions/`，清理草稿

### index.html
- 顶栏增加「📝 意见」计数按钮和「🚀 投递到 UMGEoolEx」按钮
- 新增右键菜单 `#ctx-menu`、修改意见弹窗 `#modal-overlay`、草稿列表弹窗 `#drafts-overlay`

### style.css
- 新增顶栏按钮、右键菜单、弹窗、草稿列表全套样式

### app.js
- 右键 `.nt-row` → 显示自定义菜单 → 打开带节点信息的弹窗
- 保存 / 删除 / 列出草稿的 CRUD 逻辑
- 投递全部草稿 → `POST /api/submit` → 确认提示
- 启动时刷新草稿计数

## 验证
- `node --check server.js` 语法通过
- 服务器启动正常，HTTP 200
- `POST /api/suggestions` 保存成功，`GET /api/suggestions` 返回列表
- `POST /api/submit` 生成 Markdown 到 `sessions/`，内容格式正确
- 测试数据已清理
