# UE5 MCP Tools for AI Agent to Create/Manipulate WBP (UserWidget)

## 主要推荐工具及文档

### 1. StraySpark Unreal MCP Server (200+ Tools, 包含 UMG/Widget 专用工具)
- 论坛讨论 + 工具列表: https://forums.unrealengine.com/t/strayspark-unreal-mcp-server-200-ai-tools-for-ue5-editor-automation-via-mcp/2707474
- 工具预设文档: https://www.strayspark.studio/docs/unreal-mcp-server/presets

### 2. UMGBridge Plugin (专为 UMG Widget 创建与操作的 TCP MCP 桥接)
- LobeHub MCP 页面: https://lobehub.com/mcp/977908569-umgbridgeplugin
- 相关 GitHub (UnrealMotionGraphicsMCP): https://github.com/winyunq/UnrealMotionGraphicsMCP

### 3. VibeUE (支持 UMG Widget、Blueprint 自动化 + In-Editor Chat)
- 官网: https://www.vibeue.com/
- GitHub: https://github.com/kevinpbuckley/VibeUE
- 安装文档: https://www.vibeue.com/docs/installation
- 完整文档: https://www.vibeue.com/docs

### 4. flopperam/unreal-engine-mcp (安装指南 + 示例)
- GitHub: https://github.com/flopperam/unreal-engine-mcp
- 安装视频教程: https://www.youtube.com/watch?v=ct5dNJC-Hx4

### 5. winyunq/UnrealMotionGraphicsMCP (专注 UMG 的 MCP Server)
- GitHub: https://github.com/winyunq/UnrealMotionGraphicsMCP
- 论坛讨论: https://forums.unrealengine.com/t/open-source-gemini-3-umg-an-ai-assistant-dedicated-to-ui-editing-mcp/2680786

### 6. 其他实用 MCP Server / Bridge
- jl-codes/unreal-5-mcp: https://github.com/jl-codes/unreal-5-mcp
- kvick-games/UnrealMCP: https://github.com/kvick-games/UnrealMCP
- Natfii/ue5-mcp-bridge: https://github.com/Natfii/unrealclaude-mcp-bridge
- chongdashu/unreal-mcp: https://github.com/chongdashu/unreal-mcp
- mirno-ehf/ue5-mcp (Blueprint 编辑): https://github.com/mirno-ehf/ue5-mcp

### 7. 插件形式 AI 助手（无需外部 MCP 也可创建 WBP）
- NoGlyph Blueprint AI: https://forums.unrealengine.com/t/noglyph-blueprint-ai-blueprint-ai-assistant-and-generator/2701067
